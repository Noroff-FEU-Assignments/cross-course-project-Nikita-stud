Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

I would advise them to find a suitable host that is on the more expensive side, one that provides support and security like no other. Depending on what website our client has, we can begin to look into double authentication of some sort. It could be a code send as an email or SMS additional to the password. Get yourself SSL certified to have yours and your clients data encrypted.
